# 🏏 IPL Data Analysis Dashboard (2008–2025)

## 📌 Project Overview

This project is an interactive IPL Data Analysis Dashboard developed using **Power BI**. It provides comprehensive insights into every IPL season from **2008 to 2025**, including champions, runners-up, Orange Cap holders, Purple Cap holders, team performance, and overall tournament statistics.

---

## 🛠️ Tools & Technologies

- 📊 Power BI
- 🗄️ SQL
- 📑 Microsoft Excel
- 📈 Data Visualization

## ✨ Dashboard Features

- 🏆 Season Winner Team
- 🥈 Season Runner-Up Team
- 📅 Season Filter (2008–2025)
- 🏏 Total Matches
- 👥 Total Teams
- 4️⃣ Total Fours
- 6️⃣ Total Sixes
- 🎯 Total Wickets
- 🟠 Orange Cap Holder Details
- 🟣 Purple Cap Holder Details
- 📈 Interactive Dashboard with Dynamic Filters

---

## 📌 Key Insights

- View IPL statistics for any season from 2008–2025.
- Compare winning and runner-up teams.
- Identify Orange Cap and Purple Cap winners.
- Analyze total matches, teams, fours, sixes, and wickets.
- Interactive slicer for season-wise analysis.
